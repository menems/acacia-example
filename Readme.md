## Acacia Example

This is a simple example of [acacia](https://github.com/menems/acacia)

## Installation

```bash
$ git clone https://github.com/menems/acacia-example.git
$ cd acacia-example
$ npm install
```

## Usage

```bash
$ node index.js
```
